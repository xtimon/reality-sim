# Руководство по анализу результатов симуляции

## Обзор

Скрипт `analyze_simulation.py` предназначен для комплексного анализа результатов симуляции квантовой эмерджентности. Он автоматически обрабатывает сохраненные данные и создает визуализации и статистические отчеты.

## Использование

### Базовое использование

```bash
python3 analyze_simulation.py
```

Скрипт автоматически найдет файлы `simulation_data.json` и `decoherence_data.csv` в текущей директории.

### Указание файлов

```bash
python3 analyze_simulation.py --json my_data.json --csv my_decoherence.csv
```

## Что анализируется

### 1. Квантовые системы
- Распределение по количеству кубитов
- Статистика запутанности (среднее, медиана, мин/макс, стандартное отклонение)
- Статистика когерентности
- Результаты множественных измерений
- Вероятности для каждого кубита

**Выходные файлы:**
- `quantum_systems_analysis.png` - гистограммы распределений

### 2. Декогеренция
- Временные ряды для каждого наблюдателя
- Сила декогеренции
- Время полураспада когерентности
- Сравнение наблюдателей

**Выходные файлы:**
- `decoherence_analysis.png` - графики декогеренции во времени и сравнение силы

### 3. Рождение частиц
- Зависимость от энергии вакуума
- Статистика по времени создания
- Распределение количества пар частиц

**Выходные файлы:**
- `particle_creation_analysis.png` - зависимости и распределения

### 4. Параметрические развертки
- Принцип Ландауэра: зависимость от температуры
- Принцип Ландауэра: зависимость от количества бит
- Зависимость запутанности от силы запутанности

**Выходные файлы:**
- `landauer_temperature.png`
- `landauer_bits.png`
- `entanglement_strength.png`

## Интерпретация результатов

### Запутанность
- Значение близкое к 1.0 означает высокую запутанность
- Значение близкое к 0.0 означает отсутствие запутанности

### Декогеренция
- Большая сила декогеренции → быстрое разрушение когерентности
- Время полураспада показывает, за какое время когерентность падает до 50%

### Рождение частиц
- Более высокая энергия вакуума → больше пар частиц
- Время создания показывает, когда в симуляции создаются частицы

### Принцип Ландауэра
- Энергия линейно зависит от температуры
- Энергия линейно зависит от количества бит
- Оба графика должны быть линейными в логарифмическом масштабе

## Устранение неполадок

### Ошибка "Файл не найден"
Убедитесь, что файлы `simulation_data.json` и `decoherence_data.csv` находятся в той же директории, что и скрипт, или укажите пути явно через аргументы командной строки.

### Ошибка импорта matplotlib
Установите matplotlib:
```bash
pip install matplotlib
```

Или запустите скрипт без визуализации - он все равно выполнит статистический анализ.

### Пустая папка analysis_output
Убедитесь, что у скрипта есть права на запись в текущую директорию.

done
